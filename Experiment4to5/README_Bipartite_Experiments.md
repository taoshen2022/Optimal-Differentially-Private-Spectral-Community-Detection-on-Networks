# Bipartite GapPTR Experiments

This folder contains the main-text bipartite simulation scripts. No real network needs to be downloaded or prepared. Each Monte Carlo replication generates a fresh bipartite DCSBM network internally.

## 1. Folder structure

Use the following structure:

```text
ProjectRoot/
  Utility/
    bipartite_gaptr_utils.py
  BipartiteExperiment/
    run_bipartite_vary_eps.py
    run_bipartite_vary_m.py
```

The two experiment scripts import the utility as

```python
from Utility.bipartite_gaptr_utils import ...
```

They also include a local fallback import, so they can still be tested in the same folder as `bipartite_gaptr_utils.py`.

## 2. Data-generating model

The simulation uses one main-text Bi-DCSBM setting.

We fix

\[
K=2, \qquad J=2,
\]

where `K` is the number of left-side communities and `J` is the number of right-side communities. The left-side nodes are balanced across the two communities, and the right-side nodes are also balanced across the two communities.

The baseline bipartite connectivity matrix is

\[
P_B =
\begin{pmatrix}
0.7 & 0.1 \\
0.1 & 0.7
\end{pmatrix}.
\]

The degree parameters are generated independently as

\[
\theta_i \sim \mathrm{Unif}(0.7,1.0),
\qquad
\phi_j \sim \mathrm{Unif}(0.7,1.0).
\]

Conditional on the population matrix

\[
\Omega_{ij}=\theta_i P_B(z_i,w_j)\phi_j,
\]

the observed bipartite adjacency matrix is generated by

\[
B_{ij}\sim \mathrm{Bernoulli}(\Omega_{ij})
\]

independently over all left-right node pairs.


## 3. Methods compared

For each generated network, the scripts evaluate:

| Method | Description |
|---|---|
| `NonPrivate` | Bipartite spectral clustering using the leading left singular vectors of `B` |
| `GapPTR`, `eps1=0` | GapPTR using oracle `theta0` |
| `GapPTR`, `eps1>0` | GapPTR using a private estimate of `theta0` |

The clustering error is computed on the left-side nodes after optimal label matching.

## 4. Script 1: varying total epsilon

The script

```text
run_bipartite_vary_eps.py
```

varies the total privacy budget

\[
\varepsilon_{\mathrm{total}}=\varepsilon+\varepsilon_1.
\]

Default values are

```python
eps_total_values = [0.5, 1.0, 1.5, 2.0, 2.5, 3.0, 3.5, 4.0]
eps1_values = [0.0, 0.1, 0.2]
n = 800
m = 5000
delta = 0.01
a0 = 0.10
```

Run from `BipartiteExperiment/`:

```bash
python run_bipartite_vary_eps.py \
  --n 800 \
  --m 5000 \
  --reps 30 \
  --eps1_values 0.0 0.1 0.2 \
  --outdir outputs/vary_eps_main
```

To use a custom epsilon grid:

```bash
python run_bipartite_vary_eps.py \
  --eps_total_values 0.5 1.0 1.5 2.0 2.5 3.0 3.5 4.0 \
  --n 800 \
  --m 5000 \
  --reps 30 \
  --outdir outputs/vary_eps_main
```

The script saves:

```text
outputs/vary_eps_main/
  vary_eps_raw.csv
  vary_eps_summary.csv
  vary_eps_clustering_error.png
```

## 5. Script 2: varying the number of right-side nodes

The script

```text
run_bipartite_vary_m.py
```

fixes the total privacy budget and varies the number of right-side nodes `m`.

Default values are

```python
m_values = [1000, 2000, 3000, 4000, 5000, 6000]
eps_total = 0.8
eps1_values = [0.0, 0.1, 0.2]
n = 800
delta = 0.01
a0 = 0.10
```

Run from `BipartiteExperiment/`:

```bash
python run_bipartite_vary_m.py \
  --n 800 \
  --eps_total 0.8 \
  --m_values 1000 2000 3000 4000 5000 6000 \
  --reps 30 \
  --eps1_values 0.0 0.1 0.2 \
  --outdir outputs/vary_m_main
```

The script saves:

```text
outputs/vary_m_main/
  vary_m_raw.csv
  vary_m_summary.csv
  vary_m_clustering_error.png
```

## 6. Important command-line arguments

| Argument | Meaning |
|---|---|
| `--n` | Number of left-side nodes |
| `--m` | Number of right-side nodes for the varying-epsilon experiment |
| `--m_values` | Grid of right-side node counts for the varying-`m` experiment |
| `--eps_total_values` | Grid of total privacy budgets |
| `--eps_total` | Fixed total privacy budget for the varying-`m` experiment |
| `--eps1_values` | Privacy budgets used to estimate `theta0`; `eps1=0` means oracle `theta0` |
| `--delta` | DP failure probability; default is `0.01` |
| `--a0` | Gap threshold constant; default is `0.10` |
| `--reps` | Number of Monte Carlo replications |
| `--seed_base` | Base random seed |
| `--noise_mult` | Optional multiplier for the Gaussian noise scale |
| `--outdir` | Output directory |
