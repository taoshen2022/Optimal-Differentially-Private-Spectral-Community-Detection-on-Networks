# Experiment 2 Runner

This folder contains the runner for **Experiment 2**, where GapPTR is compared with EdgeFlip and the non-private spectral clustering benchmark as the total privacy budget varies.
---

## 1. Recommended folder structure

Place the runner under the `Experiment2/` folder and keep the shared utility under `Utility/`:

```text
ProjectRoot/
  Utility/
    gapptr_utility.py
    Edge_flip.py                 # optional, needed if running EdgeFlip

  Experiment2/
    run_exp2_vary_eps_all_scenarios.py

    E2S1/
      rep000/
      rep001/
      ...

    E2S2/
      rep000/
      rep001/
      ...
```

Each replication folder should contain the saved files generated by the simulation-data utility, for example

```text
A_eigvecs_topK.csv
A_eigvals_topKp1.csv
meta.json
omega_components.npz
A_upper_bitpack.bin
```

---

## 2. Scenarios

The runner uses the following default mapping:

| Scenario | Folder | Description |
|---|---|---|
| Scenario 1 | `E2S1/` | Regular heterogeneous DCSBM setting |
| Scenario 2 | `E2S2/` | Denser setting with a low-degree-node component |

By default, both scenarios are run.

---

## 3. Privacy-budget grid

The total privacy budget is
\(
\varepsilon_{\mathrm{all}} = \varepsilon + \varepsilon_1.
\)
The default grid is
\[
\log(\varepsilon_{\mathrm{all}})
\in
\{-1,-0.75,-0.5,-0.25,0,0.25,0.5,0.75,1\}.
\]

Equivalently, in Python:

```python
log_eps_all_list = np.arange(-1.0, 1.0 + 1e-12, 0.25)
eps_all_list = np.exp(log_eps_all_list)
```

---

## 4. Methods

For each value of \(\varepsilon_{\mathrm{all}}\), the script runs:

| Method | Privacy split |
|---|---|
| `GapPTR_oracle` | \(\varepsilon=\varepsilon_{\mathrm{all}},\ \varepsilon_1=0\) |
| `GapPTR_tilde` | \(\varepsilon=\varepsilon_{\mathrm{all}}-0.2,\ \varepsilon_1=0.2\) |
| `EdgeFlip_eps_delta` | \(\varepsilon=\varepsilon_{\mathrm{all}}\) |
| `NonPrivate` | Non-private spectral clustering benchmark |

The important correction from the older script is that the tilde version now uses

```python
eps1_tilde = 0.2
eps_gap_tilde_list = eps_all_list - eps1_tilde
```

---

## 5. Basic usage

From the `Experiment2/` folder, run

```bash
python run_exp2_vary_eps_all_scenarios.py
```

This searches for data in

```text
./E2S1
./E2S2
```

and saves results to

```text
./results_exp2/
```

For running Scenario 1 only:

```bash
python run_exp2_vary_eps_all_scenarios.py --scenarios 1
```

For running Scenario 2 only:

```bash
python run_exp2_vary_eps_all_scenarios.py --scenarios 2
```

---

## 6. Output files

The default output folder is

```text
results_exp2/
```

For each scenario, the script saves a summary table:

```text
summary_exp2_s1.csv
summary_exp2_s2.csv
```

and one plot:

```text
plot_exp2_s1.png
plot_exp2_s2.png
```

It also saves replication-level diagnostic files, for example

```text
raw_exp2_s1_GapPTR_oracle.csv
raw_exp2_s1_GapPTR_tilde.csv
raw_exp2_s1_EdgeFlip_eps_delta.csv
raw_exp2_s1_NonPrivate.csv
```

Finally, it saves the combined table

```text
summary_exp2_all_scenarios.csv
```
